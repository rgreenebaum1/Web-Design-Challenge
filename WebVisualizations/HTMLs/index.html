Summary: Latitude vs. X

The purpose of this project was to analyze how the weather changes as you approach the equator. First, we extracted the data from the OpenWeatherMap API to assemple a dataset on over 450 cities.


Once the dataset was assembled, we used Matplotlib to plot several aspects of the weather vs. latitude. These are the factors we took into account: cloudiness, temperature, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and comparisons found.
